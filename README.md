Zadanie 1 - Kalkulator BMI

Program pyta użytkownika o wagę (w kg) i wzrost (w cm), a następnie oblicza BMI.

Wzór:
BMI = waga / (wzrost / 100)^2

Interpretacja:

- BMI < 18.5 - niedowaga
- 18.5 <= BMI < 25 - waga prawidłowa
- 25 <= BMI < 30 - nadwaga
- BMI >= 30 - otyłość

python zadanie1/bmi.py
