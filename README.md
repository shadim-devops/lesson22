## Zadanie 1 - Kalkulator BMI

Program pyta użytkownika o wagę i wzrost, a następnie oblicza BMI.

Wzór:
BMI = waga / (wzrost / 100)^2

Interpretacja:
- BMI < 18.5 — niedowaga
- 18.5 - 25 — waga prawidłowa
- 25 - 30 — nadwaga
- >= 30 — otyłość

Uruchomienie programu:

python zadanie1/bmi.py
